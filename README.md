# Power-BI-Dashboard
This project showcases an interactive Power BI dashboard designed to analyze and visualize comprehensive sales performance insights. Using Power BI, DAX, and Power Query, the report tracks key business metrics such as Total Sales (769M+), Units Sold (19K+), and customer behavior across multiple dimensions.

Key Features

Built a dynamic dashboard with line charts, treemaps, slicers, and interactive visuals to monitor key business KPIs.

Performed deep-dive analysis across month, product model, payment method, and customer ratings to uncover trends and patterns.

Developed DAX measures for advanced calculations and created geospatial visuals and filters for enhanced exploration and decision-making.

Tools Used

Power BI | DAX | Power Query
